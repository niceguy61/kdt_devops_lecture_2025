### 🎯 프로젝트 목표
**"Docker 기반 실무 애플리케이션 구축"**
- 보안이 강화된 멀티 컨테이너 애플리케이션
- 모니터링과 로깅이 통합된 운영 환경
- 실무에서 바로 적용 가능한 구성

1. **멀티 컨테이너**: Docker Compose 기반 3-tier 아키텍처
2. **보안 강화**: 취약점 스캔과 보안 설정 적용
3. **모니터링**: CLI 기반 실시간 모니터링 구성
4. **로그 관리**: 중앙화된 로그 수집과 분석
5. **자동화**: 배포와 모니터링 자동화 스크립트
