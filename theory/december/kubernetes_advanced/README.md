# 🚀 Kubernetes Advanced - EKS & Istio 마스터 과정

## 📅 과정 개요
**기간**: 5일 (하루 2시간, 50분 x 2세션)  
**목표**: EKS 클러스터 구축부터 Istio 서비스 메시까지 실무 중심 학습

## 🎯 학습 목표
- EKS 클러스터 생성 및 관리
- Helm을 통한 패키지 관리
- 실제 애플리케이션 배포
- Istio 서비스 메시 구성
- 마이크로서비스 트래픽 관리
