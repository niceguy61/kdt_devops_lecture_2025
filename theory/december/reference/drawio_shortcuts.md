# Draw.io 단축키 가이드

## 개요

이 문서는 Draw.io (diagrams.net)에서 효율적으로 다이어그램을 작성하기 위한 필수 단축키와 팁을 정리한 가이드입니다.

---

## 필수 단축키

### 기본 조작

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl + C` | 복사 | 선택한 요소 복사 |
| `Ctrl + V` | 붙여넣기 | 복사한 요소 붙여넣기 |
| `Ctrl + X` | 잘라내기 | 선택한 요소 잘라내기 |
| `Ctrl + D` | 복제 | 선택한 요소를 바로 복제 (가장 많이 사용!) |
| `Ctrl + Z` | 실행 취소 | 이전 작업 취소 |
| `Ctrl + Y` | 다시 실행 | 취소한 작업 다시 실행 |
| `Delete` | 삭제 | 선택한 요소 삭제 |

### 선택 및 이동

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl + A` | 전체 선택 | 캔버스의 모든 요소 선택 |
| `Shift + 클릭` | 다중 선택 | 여러 요소를 선택 |
| `Ctrl + 클릭` | 개별 추가 선택 | 선택에 요소 추가/제거 |
| `화살표 키` | 미세 이동 | 선택한 요소를 1px씩 이동 |
| `Shift + 화살표` | 빠른 이동 | 선택한 요소를 10px씩 이동 |

### 그룹화 및 레이어

| 단축키 | 기능 | 설명 |
|--------|------|------|
| `Ctrl + G` | 그룹화 | 선택한 요소들을 그룹으로 묶기 |
| `Ctrl + Shift + G` | 그룹 해제 | 그룹을 개별 요소로 분리 |

| `Ctrl + B` | 한 단계 뒤로 | 선택한 요소를 한 단계 아래로 |

## 정렬 및 배치 팁

### 1. 균등 분포

여러 요소를 선택한 후:
- **Arrange → Distribute → Horizontally**: 수평으로 균등 분포
- **Arrange → Distribute → Vertically**: 수직으로 균등 분포

### 2. 동일한 크기 만들기

여러 요소를 선택한 후:
- **Arrange → Same Size → Width**: 동일한 너비
- **Arrange → Same Size → Height**: 동일한 높이
- **Arrange → Same Size → Both**: 너비와 높이 모두 동일

### 3. 스마트 가이드 활용

- **View → Guides**: 가이드 라인 표시
- 요소를 이동할 때 자동으로 정렬선이 나타남
- 다른 요소와 정렬될 때 파란색 선으로 표시

### 4. 그리드 스냅

- **View → Grid**: 그리드 표시
- **View → Snap to Grid**: 그리드에 자동 정렬
- 정확한 배치에 유용

### 5. 컨테이너 활용

- 여러 요소를 포함하는 큰 사각형 생성
- 컨테이너를 이동하면 내부 요소도 함께 이동
- VPC, 서브넷 등을 표현할 때 유용

## 스타일 복사 방법

### 방법 1: Format Painter (권장)

1. 스타일을 복사할 원본 요소 선택
2. **Edit → Copy Style** (`Ctrl + Shift + C`)
3. 스타일을 적용할 대상 요소 선택
4. **Edit → Paste Style** (`Ctrl + Shift + V`)

### 방법 2: 스타일 패널 사용

1. 원본 요소 선택
2. 오른쪽 패널에서 스타일 확인
3. 대상 요소 선택
4. 동일한 스타일 수동 적용

### 방법 3: 기본 스타일 설정

1. 원하는 스타일의 요소 선택
2. **Edit → Set as Default Style**
3. 이후 추가되는 동일 타입 요소에 자동 적용

## 효율적인 작업 흐름

### 1. 템플릿 활용

```
1. 자주 사용하는 구조를 템플릿으로 저장
2. File → Save As Template
3. 새 다이어그램 시작 시 템플릿 선택
```

### 2. 라이브러리 관리

```
1. File → Open Library → 필요한 라이브러리 추가
2. AWS, Azure, GCP 등 클라우드 아이콘
3. 자주 사용하는 라이브러리는 즐겨찾기 추가
```

### 3. 레이어 활용

```
1. View → Layers
2. 배경, 인프라, 애플리케이션 등으로 분리
3. 레이어별로 표시/숨김 가능
```

### 4. 페이지 활용

```
1. 복잡한 시스템은 여러 페이지로 분리
2. 개요 → 상세 구조로 구성
3. 페이지 간 링크 추가 가능
```

## 실전 워크플로우

### AWS 아키텍처 다이어그램 작성

```
1. AWS 라이브러리 추가 (File → Open Library → AWS)
2. VPC 컨테이너 생성 (큰 사각형)
3. 서브넷 컨테이너 추가 (중간 사각형)
4. AWS 아이콘 드래그 앤 드롭
5. Ctrl + D로 빠르게 복제
6. Ctrl + Shift + L/R/T/B로 정렬
7. 연결선 추가 (Alt + X)
8. 레이블 추가 (F2)
9. 스타일 통일 (Ctrl + Shift + C/V)
10. 최종 확인 및 저장 (Ctrl + S)
```

### Kubernetes 다이어그램 작성

```
1. Namespace 컨테이너 생성
2. Ingress, Service, Deployment 아이콘 추가
3. Pod를 Deployment 내부에 배치
4. 연결선으로 관계 표시
5. ConfigMap, Secret 추가
6. 점선으로 참조 관계 표시
7. 색상으로 구분 (Frontend/Backend)
8. 레이블 명확히 작성
```

## 일반적인 문제 해결

### 문제 1: 요소가 정렬되지 않음
**해결**: View → Snap to Grid 활성화

### 문제 2: 연결선이 이상하게 그려짐
**해결**: 연결선 선택 → 스타일에서 "Waypoints" 조정

### 문제 3: 텍스트가 잘림
**해결**: 요소 선택 → Text → Word Wrap 활성화

### 문제 4: 그룹화된 요소를 개별 편집하고 싶음
**해결**: Ctrl + 클릭으로 그룹 내 개별 요소 선택

### 문제 5: 내보낸 이미지 품질이 낮음
**해결**: File → Export as → PNG → DPI를 300으로 설정

## 고급 팁

### 1. 커스텀 도형 만들기

```
1. 기본 도형 조합
2. Ctrl + G로 그룹화
3. Edit → Edit Style에서 세부 조정
4. 라이브러리에 저장
```

### 2. 조건부 서식

```
1. 요소 선택
2. Edit → Edit Data
3. 속성 추가 (예: status=active)
4. 스타일에서 조건부 색상 적용
```

### 3. 자동 레이아웃

```
1. Arrange → Layout → Horizontal/Vertical Flow
2. 복잡한 구조를 자동으로 정리
3. 간격 조정 가능
```

### 4. 협업 기능

```
1. File → Share → 링크 생성
2. 실시간 협업 가능
3. 댓글 기능 활용
```

```
8. Alt + X (연결선 추가)
9. Ctrl + S (저장)
```
