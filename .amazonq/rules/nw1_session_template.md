# NW1 표준 Session 템플릿 규칙

## 📋 필수 구조

초등학생도 이해할 수 있는 수준의 간단하지만 비약이 없이 깔끔한 설명이어야 합니다.

모든 Session은 다음 구조를 **반드시** 따라야 합니다:

### 1. 헤더
```markdown
# Week N Day N Session N: [서비스명]

<div align="center">
**키워드 1** • **키워드 2** • **키워드 3**
*한 줄 설명*
</div>

---
```

### 2. 세션 정보
```markdown
## 🕘 세션 정보
**시간**: HH:MM-HH:MM (50분)
**목표**: 구체적 목표
**방식**: 학습 방식
```

### 3. 학습 목표
```markdown
## 🎯 학습 목표
- 목표 1
- 목표 2
- 목표 3
```
